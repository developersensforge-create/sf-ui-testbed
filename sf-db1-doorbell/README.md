# SF-DB1 Battery Doorbell Camera — UI Prototype

Interactive clickable web prototype for the Sensforge App doorbell interface.

## How to Preview
Open `index.html` in any browser. The prototype displays a phone-sized frame (390×844px).

## Screens
1. **Live View** — Camera feed, doorbell controls (Answer/Decline/Talk/Snapshot/Privacy), event preview
2. **Doorbell Press Event** — Incoming call UI with pulsing animation, Answer/Decline, AI caption
3. **Package Detected** — Package delivery event card with sub-events and actions
4. **Events Tab** — Chronological timeline with AI-described events and thumbnails
5. **Chime Settings** — Indoor chime toggle, ringtone picker, volume, night mode
6. **Device Settings** — Full settings menu for the doorbell device
7. **Smart Detection** — Per-category detection toggles, detection zone, sensitivity slider
8. **Two-Way Audio** — Audio settings, noise cancellation, echo suppression, mic test

## Design System
- Primary accent: Orange `#F57C00`
- White backgrounds, black/gray text
- Consistent with Sensforge mobile app design language
